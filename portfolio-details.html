Quick Care — One-Tap First Aid

Quick Care is a React Native mobile app that delivers concise, media-rich first aid instructions for common emergencies. It focuses on fast discoverability, bilingual content (English & Sinhala), and offline availability for critical guidance.

The app provides step-by-step emergency procedures (CPR prompts, bleeding control, choking response, burn first aid) with short animations and video snippets. Content is optimized for quick comprehension and local caching ensures access without connectivity.

Balancing concise, reliable medical guidance with accessibility and offline resilience. Additionally, delivering video/media assets without causing heavy network usage required careful asset sizing and caching strategies.

Implemented a lightweight UI with large action buttons, concise instruction cards, and optional media. Used Firestore for content management, Firebase Storage for media, and local SQLite caching to make core instructions available offline.

Key Features

  • One-tap first aid guidance for common emergencies
  • Bilingual content (English & Sinhala)
  • Short embedded video/animation demonstrations
  • Offline caching for critical instructions
  • Bookmarking & Favorites for quick access
  • Accessible UI with large action buttons
