ServiceGo — Two-Sided Service Marketplace

ServiceGo is a Flutter-powered two-sided marketplace that connects customers with nearby service providers. It streamlines service discovery, provider management, and real-time availability tracking through GPS-based proximity sorting and an intuitive dual-module interface.

Currently, there is no unified and reliable mobile application that allows customers to quickly identify available service providers nearby or view and contact them based on service categories. Similarly, service providers lack a self-service mechanism to register, update profiles, and indicate real-time availability. This results in inefficiency, delays in service delivery, and missed opportunities for both parties.

Service Provider Module: Self-registration through an in-app form, profile creation with editable details, and availability toggle (Available/Unavailable).

Customer Module: Self-registration, profile creation, categorized view of service providers (electricians, plumbers, appliance repair), GPS-based proximity sorting (nearest to farthest within a given range), direct call button to contact providers, and add providers to a Favorites List for quick future access.

Frontend (Mobile App): Flutter (cross-platform, fast UI development)

Backend & Hosting: Firebase (serverless, lightweight backend)

Database: Firestore (NoSQL cloud database with real-time sync)

Authentication: Firebase Authentication (Email/Password)

Geolocation & Maps: Flutter packages (geolocator, google_maps_flutter)

Key Features

  • Self-registration for customers & service providers
  • Service categorization (electricians, plumbers, appliance repair)
  • GPS-based proximity sorting (nearest to farthest)
  • Direct call button for provider contact
  • Favorites list for quick provider access
  • Real-time availability toggle for providers
